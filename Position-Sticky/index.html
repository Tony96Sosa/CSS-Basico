Position Sticky

La propiedad position: sticky; proporciona 5 propiedades para modificar la posicion del elemento (top, left, right, bottom) el espacio reservado para el elemento se mantiene por lo tanto no altera el flujo del html, las dimensiones se ajustan al contenido del elemento, y lo mas importante es que podemos establecer que para cuando este en determinada posicion respecto del viewport se comporte como si estuviera establecido en position: fixed; utilizado frecuentemente cuando se hace scroll y queremos que el elemento suba pero despues se quede fijo en el top.
